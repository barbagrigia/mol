# $mol_lister

##[Online demo](http://eigenmethod.github.io/mol/#demo=mol_lister_demo)

The list of rows with lazy rendering. 

## Usage example

```
$mol_lister rows /
	< header $mol_viewer
	< bodier $mol_scroller
	< footer $mol_viewer
```
## Properties

`rows()`

Property by which we set the list of rows.
