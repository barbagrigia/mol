# $mol_barer

The component for grouping several components in one composite control.

##[Online demo](http://eigenmethod.github.io/mol/#demo=mol_barer_demo_search)

## Usage examples
```
$mol_barer
	childs /
		< stringer $mol_stringer
			value > value \
		< submitter $mol_clicker
			childs /
				\Submit
```
